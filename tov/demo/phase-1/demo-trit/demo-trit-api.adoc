[[overview]]
== Overview

The `demo-trit` programfootnote:[Binary path: `tov/demo/phase-1/demo-trit/demo-trit`] validates the libtrit three-layer architecture through 25 demonstration steps. It provides a **portable public API** that can be called by games, tutorials, and SDK examples.

[IMPORTANT]
====
*This is a demo program, not a library.* It validates libtrit works correctly. The API lets external code run specific layer tests and inspect results.
====

[NOTE]
====
*What this provides:*

* âœ“ Run all 25 demonstration steps (`demo_trit_run_all`)
* âœ“ Test individual layers (MATTER, SPACE, TIME)
* âœ“ Test Bible Rail spare state detection
* âœ“ Test 3D integration across all layers
* âœ“ Inspect pass/fail counts
* âœ“ Reset for repeated runs
====

'''

[[quick-start]]
=== Quick Start

[source,c]
----
// Link against demo-trit to use the validation API

int main(void) {
    // Run all layer tests
    int failures = demo_trit_run_all();

    if (failures == 0) {
        printf("All 25 steps passed!\n");
    }

    // Or run specific layers
    demo_trit_matter();      // MATTER layer only
    demo_trit_space();       // SPACE layer only
    demo_trit_time();        // TIME layer only
    demo_trit_bible_rail();  // Bible Rail only
    demo_trit_integration(); // 3D integration

    // Inspect results
    printf("Passed: %d\n", demo_trit_get_passed());
    printf("Failed: %d\n", demo_trit_get_failed());

    return failures;
}
