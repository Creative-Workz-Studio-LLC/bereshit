[[overview]]
== Overview

The `demo-config` programfootnote:[Binary path: `tov/demo/phase-0/demo-config/demo-config`] validates the Phase 0 TOML specifications by loading and displaying all 6 configuration categories. It provides a **portable public API** in Go that can be called by other programs.

[IMPORTANT]
====
*This is a demo program, not a library.* It proves the config loader works correctly. The API lets external code run the validation and inspect results.
====

[NOTE]
====
*What this provides:*

* âœ“ Validate all 6 config categories (`DemoConfigRunAll`)
* âœ“ Load primitives, types, schemas, contracts, bible, constants
* âœ“ Display what was loaded (names, keys, file counts)
* âœ“ Inspect pass/fail counts
* âœ“ Reset for repeated runs
====

'''

[[quick-start]]
=== Quick Start

[source,go]
----
import "creativeworkzstudio.com/bereshit/tov/demo/phase-0/demo-config"

func main() {
    // Run all category validations
    failures := DemoConfigRunAll()

    if failures == 0 {
        fmt.Println("All 6 categories validated!")
    }

    // Inspect results
    fmt.Printf("Passed: %d\n", DemoConfigGetPassed())
    fmt.Printf("Failed: %d\n", DemoConfigGetFailed())

    // Reset for another run
    DemoConfigReset()
}
