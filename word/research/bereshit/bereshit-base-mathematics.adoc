== Overview

This document provides the *formal mathematical specification* for the Bereshit base algorithms. While the companion document (link:bereshit-base-algorithms.adoc[bereshit-base-algorithms.adoc]) describes HOW to compute, this document specifies WHAT the mathematical relationships ARE.

[IMPORTANT]
====
*Mathematics describes RELATIONSHIPS. Algorithms describe PROCEDURES.*

stem:[f: A \to B] says "a function from set A to set B exists" â€” it doesn't say how to compute it.

The algorithm says "take input, do step 1, do step 2, return output."

Both are necessary. Neither is sufficient alone.
