The `config` packagefootnote:[Package path: `creativeworkzstudio.com/bereshit/word/work/pkg/config`] provides a Go API for loading and validating Phase 0 TOML specifications. It reads configuration files from `word/core/` and returns structured data for use in Bereshit applications.
