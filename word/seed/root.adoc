[[overview]]
== Overview

Word Seeds is a child of word/, containing σπέρμα (sperma/seeds) — template files for non-OmniCode formats. This includes templates for Go, AsciiDoc, Markdown, configuration files, and other formats used in the bereshit system.

This document follows the CWSfootnote:[CWS = CreativeWorkzStudio LLC] 5-block documentation structure.

[IMPORTANT]
====
*#Non-OmniCode templates#* — These seeds produce files in various formats (Go, AsciiDoc, YAML, etc.). For OmniCode-specific templates (.omni files), see `word/omni/seed/`. Guided strictness — adapt templates thoughtfully while maintaining structural integrity.
====

[NOTE]
====
*Scope:* This document covers the word/seed/ folder containing non-OmniCode templates. Organized by category: code/, data/, documentation/.
====

Word Seeds organize templates by category for easy discovery and use.

----
flowchart TD
    S[word/seed/] --> C[code/]

    O --> AsciiDoc & Markdown
