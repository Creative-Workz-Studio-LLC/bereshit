[[overview]]
== Overview

OmniCode is a child of word/, containing the λόγος (logos/word) — the OmniCode programming language specification that defines what things ARE. This is the core definition language for Kingdom Technology.

This document follows the CWSfootnote:[CWS = CreativeWorkzStudio LLC] 5-block documentation structure.

[IMPORTANT]
====
*#OmniCode is the λόγος#* — It IS the Word that defines. Just as the λόγος brought creation into being through speaking, OmniCode defines the structures and patterns that implementations realize. Language specification is non-negotiable — tight strictness.
====

[NOTE]
====
*Scope:* This document covers the word/omni/ folder containing OmniCode language definition. For OmniCode templates, see `word/omni/seed/`. For non-OmniCode templates, see `word/seed/`.
====

[[structure]]
== Structure & Flow

OmniCode defines what IS. Implementations in work/ realize these definitions.

[[flow]]
=== The Logos Flow

[mermaid]
----
flowchart LR
    O[word/omni/] --> W[work/]
    W --> F[fruit/]
    style O fill:#gold
----

. *word/omni/* (λόγος) — *Definition*. What things ARE.

. *work/* (ἔργον) — *Implementation*. Realizations of definitions.

. *fruit/* (καρπός) — *Output*. Products of implementations.

[TIP]
====
*The λόγος precedes creation.* "In the beginning was the Word" — definitions come first, then implementations, then outputs. OmniCode IS the beginning.
====
