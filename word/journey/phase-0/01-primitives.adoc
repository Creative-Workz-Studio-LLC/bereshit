[[process]]
== Process: How We Built

=== 1. Identify Atomic Types

From `bereshit-base-mathematics.adoc`, we extracted the primitive types — types that cannot be broken down further:

[cols="2,4",options="header"]
|===
| Type | Mathematical Definition

| *Trit*
| Single balanced ternary digit: stem:[\{-1, 0, +1\}]

| *HealthScore*
| Storage domain: stem:[\mathbb{S} = \{0, \ldots, 255\}]

| *TrueValue*
| True domain: stem:[\mathbb{T} = \{-100, \ldots, 100\}]

| *Direction*
| Temporal dimension: stem:[\mathbb{D} = \{-1, 0, +1\}]

| *Level*
| State names: stem:[\mathbb{L}] (7 values: Dead, Critical, Poor, Fair, Good, Excellent, Perfect)

| *HardPoint*
| Coarse anchors: stem:[\mathbb{H}_5 = \{-100, -50, 0, 50, 100\}]
|===

=== 2. Create primitives.toml

Using the 3-block data template:

[source,bash]
----
cp word/seed/data/toml/toml-data.toml word/core/primitives.toml
----

=== 3. Define Each Primitive

For each type, we specified:

* **Name**: The type identifier (e.g., `Trit`, `HealthScore`)
* **Domain**: Mathematical set it represents
* **Storage**: How it's stored (bytes, bits)
* **Validation**: Constraints that make it valid (ranges, allowed values)

=== 4. Ground in Scripture

Each primitive traces to biblical source:

[cols="2,4",options="header"]
|===
| Primitive | Biblical Grounding

| *Trit*
| Genesis 1:6 — Division creating thirds (firmament dividing waters)

| *HealthScore*
| Proverbs 11:1 — Just weights (accurate measurement)

| *Level*
| Biblical state language (Psalm 23:1 "wanting" → Psalm 18:30 "perfect")

| *Direction*
| Ecclesiastes 3:1-8 — Time for every purpose (past, present, future)
|===

[[output]]
== Output: What We Produced

[source]
----
word/core/primitives.toml   # ~22KB, all atomic types defined
----

The file contains:

* **METADATA block**: Identity, purpose, biblical grounding
* **CONTENT block**: Type definitions with domains, storage, validation
* **CLOSING block**: Validation rules, relationships

=== Key Types Defined

[source,toml]
----
[primitives.trit]
name = "Trit"
domain = "{-1, 0, +1}"
storage = "2 bits (packed: 5 trits per byte)"
biblical_anchor = "Genesis 1:6"

[primitives.health_score]
name = "HealthScore"
domain = "0..255"
storage = "1 byte (uint8)"
biblical_anchor = "Proverbs 11:1"
----
