[[process]]
== Process: How We Built

=== 1. Identify Composition Patterns

How do primitives combine? We identified four primary patterns:

[cols="2,4,3",options="header"]
|===
| Pattern | Description | Example

| *Struct*
| Named collection of fields
| Health = {score: HealthScore, level: Level}

| *Array*
| Fixed or variable length sequences
| TritArray = Trit[n]

| *Map*
| Key-value associations
| Symptoms = Map<string, HealthScore>

| *Optional*
| Nullable wrapper types
| MaybeLevel = Level?
|===

=== 2. Define Type Relationships

Types form a dependency hierarchy:

[source]
----
                  ┌─────────────────┐
                  │  primitives.toml │  ← Foundation
                  └────────┬────────┘
                           │
           ┌───────────────┼───────────────┐
           │               │               │
           ▼               ▼               ▼
    ┌───────────┐   ┌───────────┐   ┌───────────┐
    │  Health   │   │ TritArray │   │  Identity │
    │  struct   │   │   array   │   │   struct  │
    └───────────┘   └───────────┘   └───────────┘
           │
           ↓
    Depends on: HealthScore, Level (primitives)
----

=== 3. Create types.toml

Using the 3-block data template:

[source,bash]
----
cp word/seed/data/toml/toml-data.toml word/core/types.toml
----

=== 4. Define Composition Rules

For each composite type, we specified:

* **Name**: The type identifier
* **Composition**: How it combines primitives
* **Fields**: What primitives/types it contains
* **Dependencies**: What it requires to exist

=== 5. Ground in Scripture

_"Let us make man in our image"_ — God created through composition and relationship. Man is composed of dust (primitive) + breath of life (composition). Types compose primitives just as man is composed of elements.

=== Key Types Defined

[source,toml]
----
[types.health]
name = "Health"
composition = "struct"
fields = [
    { name = "score", type = "HealthScore" },
    { name = "level", type = "Level" },
    { name = "direction", type = "Direction" }
]
depends_on = ["primitives.health_score", "primitives.level", "primitives.direction"]

[types.trit_array]
name = "TritArray"
composition = "array"
element = "Trit"
depends_on = ["primitives.trit"]
----
