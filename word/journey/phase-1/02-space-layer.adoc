== Purpose

This document describes *how we built dimension.h* — the SPACE layer of libtrit. Dimensions give MEANING to trit values, enabling CPI-SI to navigate through cognitive space.

[[process]]
== Process: How We Built

=== 1. Define Cognitive Dimensions

From Genesis 1:1, three dimensions of awareness:

[source,c]
----
typedef enum {
    DIM_MATERIAL = TRIT_NEG,   // -1: "the earth" - what
    DIM_TEMPORAL = TRIT_ZERO,  //  0: "in the beginning" - when
    DIM_SPATIAL  = TRIT_POS    // +1: "the heavens" - where
} dimension_t;
----

*Question aliases:*
[source,c]
----
#define DIM_WHEN   DIM_TEMPORAL    // "when" - sequence
#define DIM_WHERE  DIM_SPATIAL     // "where" - relationships
#define DIM_WHAT   DIM_MATERIAL    // "what" - identity
----

=== 2. Define Navigation Directions

From the building block method:

[source,c]
----
typedef enum {
    DIR_BREAK_DOWN = TRIT_NEG,   // -1: Toward matter/foundation
    DIR_ANCHOR     = TRIT_ZERO,  //  0: Rest at anchor/halt
    DIR_BUILD_UP   = TRIT_POS    // +1: Toward space/meaning
} direction_t;
----

*Operation aliases:*
[source,c]
----
#define DIR_DOWN   DIR_BREAK_DOWN  // Toward foundation
#define DIR_CENTER DIR_ANCHOR      // Rest at anchor
#define DIR_UP     DIR_BUILD_UP    // Toward meaning
----

=== 3. Design Navigation Functions

*Conversion between trit and dimension:*
[source,c]
----
dimension_t trit_to_dimension(trit_t t);
trit_t dimension_to_trit(dimension_t dim);
----

*Navigation through space:*
[source,c]
----
trit_t trit_navigate(trit_t current, direction_t dir);
direction_t dimension_path(dimension_t from, dimension_t to);
----

*Semantic queries:*
[source,c]
----
const char* dimension_question(dimension_t dim);  // "when", "where", "what"
const char* dimension_phrase(dimension_t dim);    // Genesis 1:1 phrases
----

=== 4. Implement src/dimension.c

Navigation with clamping:
[source,c]
----
trit_t trit_navigate(trit_t current, direction_t dir) {
    if (dir == DIR_ANCHOR) return current;

    int result = current + dir;
    if (result > TRIT_POS) return TRIT_POS;
    if (result < TRIT_NEG) return TRIT_NEG;
    return (trit_t)result;
}
----

Pathfinding between dimensions:
[source,c]
----
direction_t dimension_path(dimension_t from, dimension_t to) {
    if (from == to) return DIR_ANCHOR;
    return (to > from) ? DIR_BUILD_UP : DIR_BREAK_DOWN;
}
----
