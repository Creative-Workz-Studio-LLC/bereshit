[[emergence]]
== What Emerged

Building the three layers revealed patterns not explicitly designed:

=== 1. The +1 Shift Is Universal

The balanced-to-unsigned conversion (+1) appears everywhere:

* Pack: `(trit + 1)` shifts -1,0,+1 to 0,1,2
* Unpack: `(value % 3) - 1` reverses it
* Same pattern: MATTER→TIME→SPACE is a +1 shift conceptually

=== 2. Spare States Were Inevitable

3^5 = 243 < 2^8 = 256 isn't arbitrary — balanced ternary naturally creates "leftover" space in binary storage. These 13 spare states became Bible Rail.

=== 3. Navigation Is Clamped

Moving through dimensions doesn't wrap — it clamps:
- BUILD_UP from SPACE (+1) stays at SPACE
- BREAK_DOWN from MATTER (-1) stays at MATTER

This reflects boundaries, not loops.

=== 4. Compound States Create Depth

9 modes from 3×3 (minus self-loops) wasn't planned — it emerged from asking "What if consciousness looks FROM one time AT another?"

=== Detect-Assess-Recover

Testing follows the building block method:

[cols="2,4"]
|===
| Phase | What It Does

| *Detect*
| Tests surface when code drifts from design

|===

=== Test Categories

[cols="2,4"]
|===
