[cols="2,4,4",options="header"]
|===
| Aspect | Schema (schemas/) | Contract (contracts/)

| *Question answered*
| WHAT is valid?
| HOW must it behave?

| *Type*
| Declarative (structure)
| Imperative (behavior)

| *Example*
| "HealthScore: 0-255 integer"
| "MUST normalize before storing"

| *Validation*
| Data structure
| Operation behavior

| *When checked*
| At data boundary
| At operation boundary
|===

*The relationship:* Schemas define what valid data looks like. Contracts define what valid operations do with that data.
