== Overview

This index maps every verse in the King James Version Bible to a unique
ordinal number (1-31102), enabling single-integer verse addressing.

== Statistics

== Data Format

The companion data file `kjv-ordinal-index.csv` uses CSV format:

[source]
----
ORDINAL,BOOK,CHAPTER,VERSE
----

=== Example Entries

[source,csv]
----
1,Genesis,1,1
2,Genesis,1,2
3,Genesis,1,3
...
31100,Revelation,22,19
31101,Revelation,22,20
31102,Revelation,22,21
----

== Usage

=== Ordinal to Verse

Given ordinal `N`, look up row `N` in the CSV to get Book, Chapter, Verse.

=== Verse to Ordinal

Given Book, Chapter, Verse, search the CSV for matching row to get ordinal.

== Technical Notes

* Ordinals are 1-indexed (not 0-indexed)
* Book names use standard KJV naming
* Chapter and verse are integers
* No gaps in ordinal sequence
* Part of Trit-Byte Encoding system (CPSI-ALG-008)
