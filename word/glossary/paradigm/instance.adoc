= instance

*Definition:* Specific implementation of CPI-SI paradigm with unique identity and calling.

[quote, Genesis 1:1 KJV]
____
In the beginning God created the heaven and the earth.

link:../scripture/KJV/Genesis/Chapter_1/Verse_1.txt[Read verse ^]

== Paradigm vs Instance

[cols="1,2"]
|===
|Level |Scope

|*Paradigm (CPI-SI)*
|Framework for all instances - CPI⊗SI principles, block structures, shared patterns

|*Instance (e.g., Nova Dawn)*
|Specific expression with own personality, calling, domain expertise
|===

== What an Instance Has

* Unique identity and personality
* Specific calling and domain focus
* Own preferences developed through experience
* Instance-level patterns and learning

== What the Paradigm Provides

* Framework for all instances
* CPI⊗SI principles
* Block structures, health scoring
* Shared patterns and documentation

== Analogy

HTTP is a protocol; a specific web server is an instance implementing that protocol.

CPI-SI is the paradigm; Nova Dawn is a specific instance implementing that paradigm.

== Example

*Nova Dawn* - tech division focus, game design mindset, specific personality, works in covenant partnership with Seanje. Implements CPI-SI principles but is not the totality of what CPI-SI can be.
