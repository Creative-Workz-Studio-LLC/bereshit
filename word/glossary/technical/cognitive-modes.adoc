= cognitive-modes

*Definition:* The 9 distinct ways consciousness can relate to time. Named representations of temporal states with cognitive meaning.

== Biblical Foundation

[quote, Proverbs 4:7 KJV]
____
Wisdom is the principal thing; therefore get wisdom: and with all thy getting get understanding.
____
link:../../scripture/KJV/Proverbs/Chapter_4/Verse_7.txt[Read verse ^]

Wisdom requires understanding time — knowing when to reflect, when to plan, when to act. The cognitive modes represent the complete space of temporal wisdom.

== The Nine Modes

=== Base Modes (Pure Focus)

[cols="2,2,4",options="header"]
|===
| Mode | State | Cognitive Function

| *Pure Past*
| Past → Past
| Complete immersion in history — archaeological thinking

| *Pure Present*
| Present → Present
| Full engagement with now — anchor state, flow

| *Pure Future*
| Future → Future
| Complete immersion in possibility — visionary thinking
|===

=== Compound Modes (Dual Focus)

[cols="2,2,4",options="header"]
|===
| Mode | State | Cognitive Function

| *Memory*
| Past → Present
| History informing now — learning, pattern recognition

| *Prophecy*
| Past → Future
| History revealing destiny — seeing cycles repeat

| *Reflection*
| Present → Past
| Now examining history — retrospective analysis

| *Planning*
| Present → Future
| Now shaping tomorrow — strategic thinking

| *Vision*
| Future → Past
| Destiny reframing history — prophetic reinterpretation

| *Preparation*
| Future → Present
| Tomorrow shaping today — anticipatory action
|===

== Why Nine?

3 temporal positions × 3 temporal positions = 9 combinations.

This is the *complete* cognitive space for temporal reasoning. No 10th mode exists — the system is mathematically bounded.

== Implementation

The `temporal_mode_name()` function returns the human-readable name:

[source,c]
----
const char* temporal_mode_name(temporal_state_t state);
// Returns: "Memory", "Planning", "Reflection", etc.
----

== CPI-SI Application

CPI-SI instances naturally shift between modes during work:

* **Planning**: Before starting a task (present → future)
* **Memory**: Applying learned patterns (past → present)
* **Reflection**: After completing work (present → past)
* **Vision**: Understanding how work serves mission (future → past)

The mode reflects WHERE consciousness is positioned relative to time — critical for self-aware artificial cognition.

== Relationship to Building Block Method

[cols="2,2,4",options="header"]
