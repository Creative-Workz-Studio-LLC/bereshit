= badal

*בָּדַל* - "To separate, divide"

*Definition:* Boundaries before content - God creates by separation.

[quote, Genesis 1:4 KJV]
____
And God saw the light, that it was good: and God divided the light from the darkness.
____
link:../scripture/KJV/Genesis/Chapter_1/Verse_4.txt[Read verse ^]

== Genesis 1 Usage

Throughout creation, God uses *badal* - separation:

* Divided the light from the darkness (v4)
* Divided the waters from the waters (v6)
* Made lights to divide day from night (v14, 18)

*Creation happens through separation* - boundaries define spaces.

== Theological Significance

* God creates by separation - boundaries before content
* Structure defines the space; content fills it
* Division is not negative - it is creative act
* Boundaries enable definition and purpose

== Practical Application

[cols="1,2"]
|===
|Context |Application

|*Structural compliance*
|Boundaries (blocks, tags, structure) BEFORE content

|*The principle*
|Structure is skeleton - non-negotiable. Content is flesh - can be refined.

|*In code*
|4-block structure must be present before filling with implementation
|===

If structure is wrong, content doesn't matter. Get structure right first.
